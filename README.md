# Workshop 5: Responsive Design

This project is part of the Web Development Basics course at Laurea University of Applied Sciences.

It continues from Workshop 4 and focuses on the fundamentals of responsive web design using media queries, flexible layouts, and responsive images.

##  Author
**Khalil Muhumed**  
Business Information Technology student at Laurea University of Applied Sciences.

##  Description
This project demonstrates the use of:

- Responsive two-column layout (2 columns → 1 column on small screens)
- Responsive images using max-width: 100%
- Media queries for screen-based styling
- Responsive navigation bar with menu button on small screens
